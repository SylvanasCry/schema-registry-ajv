# schema-registry-ajv

An Ajv instance wrapper for @kafkajs/confluent-schema-registry.

## Installation

Install the library:
```shell
npm install @sylvanas-cry/schema-registry-ajv
```

Install peer dependencies:
```shell
npm install \
  @kafkajs/confluent-schema-registry \
  ajv \
  ajv-formats \
  axios
```
